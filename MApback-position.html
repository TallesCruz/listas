ol (Lista ordenada)

Indica uma sequência de itens dentro do documento HTML

  • background-size

    1. Sintaxe


      /* Ocupa todo espaço do elemento e exibe parte da imagem se for necessário */
      /* Estica a imagem ao máximo possível exibindo a imagem por completo */
      background-size: contain;
      /* Define o percentual de largura e altura com relação ao elemento, que a imagem vai ocupar */
      background-size: 50% 70%;
      /* Define a largura e altura da imagem em pixel */
      background-size: 300px 150px;

  • background-position

    1. Sintaxe


      background-position: top;
      background-position: center;
      background-position: bottom;
      background-position: right top;
      background-position: right center;
      background-position: right bottom;
      background-position: center top;
      background-position: center;
      background-size: cover;
      background-position: center bottom;

  • Atributo start

    É o ponto de partida da Lista

    Sintaxe


    <ol start="3">
    <li>Passo1</li>
    <li>Passo2</li>
    <li>Passo3</li>
    </ol>

  • Atributo reversed

    A ordem da lista será invertida

    Sintaxe


    <ol reversed>
    <li>Passo1</li>
    <li>Passo2</li>
    <li>Passo3</li>
    </ol>


ul (Lista não ordenada)

Agrupa itens dentro do documento HTML

  • Sintaxe

    1. Sintaxe


      <ul>
      <li>Algo1</li>
      <li>Algo2</li>
      <li>Algo3</li>
      </ul>

  • Atributo de Marcação (type)

    É o tipo de marcação que será usado

    1. <ul type="disc">

      2. Ordenação bolinha cheia (padrão)

    2. <ul type="square">

      3. Ordenação quadrado

    3. <ul type="circle">

      4. Ordenação bolinha vazia

    4. <ul type="none">

      5. Ordenação sem nada (vazia)
